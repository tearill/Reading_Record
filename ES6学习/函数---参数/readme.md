# 函数的参数
1. 参数扩展/数组展开
2. 默认参数

- 参数扩展：  
  1. 收集剩余的参数  
    `function show(a, b, ...args){};`
    * Rest Parameters 必须是最后一个
  2. 展开数组  
     展开后的效果，跟直接把数组的内容写在这儿一样  

---------------------------------------------------

- 默认参数： (扩展 --- 文件切片的大小)
$('#div1').animate({width: '200px'});
$('#div1').animate({width: '200px'}, 1000);
